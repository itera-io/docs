## Managing Users

### Add Users

<Icon icon="user-tie" iconType="solid" /> Manager | <Icon icon="handshake" iconType="solid" /> Partner

To add a user to your account:
1. Navigate to the *Users* section
2. Click "*Add User*" in the upper-right corner
3. Provide the required information:
   - Username
   - Display name (optional)
   - Email address
   - Organization (Partner only)
   - Role (Partner and Manager)
4. Click "+ Add" to create the profile

<Frame caption="Add Users">
  ![Add User Interface](/Documentation/Images/Account%20Management/add.user.webp)
</Frame>

### Delete Users

<Icon icon="user-tie" iconType="solid" /> Manager | <Icon icon="handshake" iconType="solid" /> Partner

To delete a user:
1. Go to the *Users* section
2. Find the user you want to remove
3. Click the "*Delete*" option
4. Confirm the deletion when prompted
