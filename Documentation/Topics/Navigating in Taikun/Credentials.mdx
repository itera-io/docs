# Backup Credentials

Taikun’s Backup system will allow you to back up your Project data to any S3-compatible storage offered on the market. The Backup Credentials tab is where you can connect your Backup storage to ensure that the data stays safe.

<Card title="Backup system" icon="right-to-bracket" href="/content/components/card-group" horizontal>
  You can learn more about our Backup system HERE
</Card>

Up here, you will find a tabs with parameters of connected storage that involve:

<Tabs>

  </Tab>

# Billing credentials

These credentials are required to ensure timely payments for cloud resources like computing power, storage, and networking. Cloud providers also use them to generate invoices and manage subscription plans.

To add billing credentials, users need to provide the following details in the “Add Credentials” dialog:

<Tabs>
  <Tab title="Organization">
    Select your organization
  </Tab>

  <Tab title="Name">
    Specify a name for your credentials (e.g., “billing”)
  </Tab>

  <Tab title="Username">
    Enter the billing service username (e.g., “user”)
  </Tab>

  <Tab title="Password">
    Enter the service password
  </Tab>

  <Tab title="URL">
    Provide the service URL (e.g., https://prometheus.####.###).
  </Tab>
</Tabs>

This information is required to configure and authenticate billing for cloud services.
