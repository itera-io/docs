# Supported Cloud Providers

Taikun currently supports six primary Cloud services: AWS, Azure, GCP, OpenStack, VMWare Tanzu, Openshift, VMware vSphere, Zadara and Proxmox. You must add your cloud credentials to the application to use Taikun to establish and manage your Kubernetes clusters on these platforms. More details about Cloud Credentials are in the Supported Cloud Providers section.

# Add Cloud Credentials

To add your Cloud credentials in Taikun follow these steps:

<Steps>
  <Step title="First Step">
    Log in to your Taikun account and navigate to the *Cloud credentials* section in the left-hand navigation panel.
  </Step>

  <Step title="Second Step">
    Click on the "Add Cloud Credentials" in top right corner.&#x20;
  </Step>

  <Step title="Third Step">
    Select the Cloud provider you want to add credentials for.
  </Step>

  <Step title="Fourth Step">
    Enter your Cloud provider’s parameters and any other required information specific to each provider
  </Step>

  <Step title="Fifth Step">
    Click on the "Add Cloud Credentials" to save your credentials.
  </Step>

  <Step title="Sixth Step">
    You should now see your new credentials listed on the page
  </Step>
</Steps>

Once you’ve added your Cloud credentials, you can use Taikun to establish and manage Kubernetes clusters on the respective cloud provider.
