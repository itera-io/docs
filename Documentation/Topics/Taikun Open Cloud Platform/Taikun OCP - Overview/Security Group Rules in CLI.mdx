# Security Group Rules in CLI



# security group rule[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#security-group-rule "Permalink to this headline")

A **security group rule** specifies the network access rules for servers and other resources on the network.

Compute v2, Network v2

## security group rule create[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#security-group-rule-create "Permalink to this headline")

Create a new security group rule

openstack security group rule create
&#x20;   \[--extra-property type=\<property\_type>,name=\<property\_name>,value=\<property\_value>]
&#x20;   \[--remote-ip \<ip-address> | --remote-group \<group>]
&#x20;   \[--dst-port \<port-range>]
&#x20;   \[--protocol \<protocol>]
&#x20;   \[--description \<description>]
&#x20;   \[--icmp-type \<icmp-type>]
&#x20;   \[--icmp-code \<icmp-code>]
&#x20;   \[--ingress | --egress]
&#x20;   \[--ethertype \<ethertype>]
&#x20;   \[--project \<project>]
&#x20;   \[--project-domain \<project-domain>]
&#x20;   \<group>


–extra-property type=\<property\_type>,name=\<property\_name>,value=\<property\_value>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-extra-property "Permalink to this definition")

Additional parameters can be passed using this property. Default type of the extra property is string (‘str’), but other types can be used as well. Available types are: ‘dict’, ‘list’, ‘str’, ‘bool’, ‘int’. In case of ‘list’ type, ‘value’ can be semicolon-separated list of values. For ‘dict’ value is semicolon-separated list of the key:value pairs.

–remote-ip \<ip-address>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-remote-ip "Permalink to this definition")

Remote IP address block (may use CIDR notation; default for IPv4 rule: 0.0.0.0/0, default for IPv6 rule: ::/0)

–remote-group \<group>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-remote-group "Permalink to this definition")

Remote security group (name or ID)

–dst-port \<port-range>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-dst-port "Permalink to this definition")

Destination port, may be a single port or a starting and ending port range: 137:139. Required for IP protocols TCP and UDP. Ignored for ICMP IP protocols.

–protocol \<protocol>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-protocol "Permalink to this definition")

*Network version 2:*

IP protocol (ah, dccp, egp, esp, gre, icmp, igmp, ipv6-encap, ipv6-frag, ipv6-icmp, ipv6-nonxt, ipv6-opts, ipv6-route, ospf, pgm, rsvp, sctp, tcp, udp, udplite, vrrp and integer representations \[0-255] or any; default: any (all protocols))

*Compute version 2:*

IP protocol (icmp, tcp, udp; default: tcp)

–description \<description>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-description "Permalink to this definition")

Set security group rule description

*Network version 2 only*

–icmp-type \<icmp-type>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-icmp-type "Permalink to this definition")

ICMP type for ICMP IP protocols

*Network version 2 only*

–icmp-code \<icmp-code>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-icmp-code "Permalink to this definition")

ICMP code for ICMP IP protocols

*Network version 2 only*

–ingress[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-ingress "Permalink to this definition")

Rule applies to incoming network traffic (default)

*Network version 2 only*

–egress[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-egress "Permalink to this definition")

Rule applies to outgoing network traffic

*Network version 2 only*

–ethertype \<ethertype>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-ethertype "Permalink to this definition")

Ethertype of network traffic (IPv4, IPv6; default: based on IP protocol)

*Network version 2 only*

–project \<project>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-project "Permalink to this definition")

Owner’s project (name or ID)

*Network version 2 only*

–project-domain \<project-domain>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-project-domain "Permalink to this definition")

Domain the project belongs to (name or ID). This can be used in case collisions between project names exist.

*Network version 2 only*

group[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-create-arg-group "Permalink to this definition")

Create rule in this security group (name or ID)

–sort-column SORT\_COLUMN[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-list-sort-column "Permalink to this definition")

*Network version 2 only*

–ethertype \<ethertype>[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-list-ethertype "Permalink to this definition")

List rules by the Ethertype (IPv4 or IPv6)

*Network version 2 only*

–ingress[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-list-ingress "Permalink to this definition")

List rules applied to incoming network traffic

*Network version 2 only*

–egress[¶](https://taikun.cloud/docs/security-group-rules-in-cli/#cmdoption-openstack-security-group-rule-list-egress "Permalink to this definition")

*Network version 2 only*
