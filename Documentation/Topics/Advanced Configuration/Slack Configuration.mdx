# Add Slack Configuration

First, you need to set your webhooks for Slack and create a Slack channel where you want to receive notifications.

<Frame caption="Slack Webhook">
  ![](https://mintlify.s3-us-west-1.amazonaws.com/taikuncloud/Documentation/Images/Advanced%20Configuration/slack.2.webp)
</Frame>

## Fill in the following parameters:

<AccordionGroup>
  <Accordion title="Organization" icon="sitemap">
    Choose an Organization to which you want to add Slack notifications
  </Accordion>

  <Accordion title="Name" icon="pencil">
    Select a name for your configuration
  </Accordion>

  <Accordion title="URL" icon="globe">
    Insert webhook URL from the Slack app; [this Slack guide can be of help](https://dashboard.mintlify.com/taikuncloud/taikuncloud/editor#url)
  </Accordion>

  <Accordion title="Channel" icon="slack">
    Select Slack channel for receiving notifications from Taikun
  </Accordion>

  <Accordion title="Type" icon="bell">
    Alert (receive only alert-type of notifications) or General (receive all notifications)
    Now you will receive notifications in the selected Slack channel from your Taikun Projects!
  </Accordion>
</AccordionGroup>

<Frame caption="Slack Channel">
  ![](https://mintlify.s3-us-west-1.amazonaws.com/taikuncloud/Documentation/Images/Advanced%20Configuration/slack-app.png)
</Frame>

<Info>Slack notification will contain a link for direct access to your Taikun Project</Info>
