# Getting Started with Taikun CLI

<Steps>
  <Step title="First Step">
    To download the CLI, head to [the latest release page](https://github.com/itera-io/taikun-cli/releases/tag/v3.4.1)
  </Step>

  <Step title="Second Step">
    Scroll down to the Assets section and select the binary for your architecture
  </Step>

  <Step title="Third Step">
    Use the sudo cp taikun /usr/local/bin/ command to move the binary
  </Step>
</Steps>

Signing in to Taikun The Taikun CLI reads environment variables to authenticate to Taikun.

## To authenticate with your Taikun account:

<AccordionGroup>
  <Accordion title="set the following environment variables:">
    ```
    TAIKUN_EMAIL
    TAIKUN_PASSWORD
    ```
  </Accordion>

  <Accordion title="To authenticate with Keycloak, set the following environment variables:">
    ```
    TAIKUN_KEYCLOAK_EMAIL
    AIKUN_KEYCLOAK_PASSWORD
    ```
  </Accordion>
  
  <Accordion title="The default API host is api.taikun.cloud. To override it, set the following environment variable:">
    ```
    TAIKUN_API_HOST (default value is: api.taikun.cloud)
    ```
  </Accordion>
  <Accordion title="Run the following command to check whether you are properly authenticated:">
    ```
    taikun whoami
    ```
  </Accordion>
</AccordionGroup>
